Project Overview

SmartTrip is a smart travel itinerary planner designed to help users plan their trips efficiently by generating day-by-day travel plans.

Key Features

  • Destination-based trip planning
  • Day-wise itinerary generation
  • User-friendly and responsive UI
  • Future map and collaboration features

Technologies Used

  • HTML5
  • CSS3
  • Git & GitHub
  • JavaScript (planned)
